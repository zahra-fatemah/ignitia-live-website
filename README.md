# 🔥 IGNITIA'26 – Official Event Platform

A full-stack event management platform built for IGNITIA'26, the flagship techno-cultural fest of IEM-UEM Group, Kolkata.

---

## 🚀 Features

- 🔐 Secure Authentication (Email/Password)
- 🎯 Event Registration System
- 👥 Team Creation & Join via Code
- 📊 User Dashboard (Registrations, Teams)
- 🛠️ Admin Panel (Manage Events & Users)
- 📩 Contact Form Integration
- ⚡ Fast & Responsive UI

---

## 🧑‍💻 Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS

### Backend
- Supabase (Auth + Database + RLS)

## ⚙️ Setup Instructions

### 1. Clone the repository
git clone https://github.com/your-username/ignitia-live-website.git
cd ignitia-live-website

2. Install dependencies
npm install
3. Run locally
npm run dev

🌐 Live Demo
👉 http://localhost:4173/

🔐 Environment Variables
Create a .env file and add:
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
